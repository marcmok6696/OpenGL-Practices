Fan Rotating:
Key"z": Rotate the fan clockwise
Key"x": Rotate the fan anticlockwise

Gem Scaling:
Key"r": Fine Scaling down the gem
Key"t": Fine Scaling up the gem
Left Mouse Click: Coarse scaling down the gem
Left Mouse Click: Coarse scaling up the gem

House Translation:
Key"i": Translate the house backward
Key"o": Translate the house forward

Pyramid Translation:
Key"w": Translate the pyramid upwards
Key"a": Translate the pyramid leftwards
Key"d": Translate the pyramid rightwards
Key"s": Translate the pyramid downwards
